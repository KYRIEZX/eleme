## 仿饿了么webApp

## 技术栈
>vue, vue-router4, vuex, mint-ui, bootstrap, betterScroll

## 项目运行

```
$ git clone https://github.com/KYRIEZX/eleme.git
$ npm install
$ npm run serve
```

## 基本功能
>登录页面、店铺列表、分类筛选、定位搜索、城市选择等
